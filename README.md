# Task Manager (To-Do App)

A modern and responsive Task Manager web application built using:
- HTML
- CSS (Flexbox)
- JavaScript

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Search Tasks
- Task Description
- Task Deadline / Timeline
- Mark as Completed
- Smooth Animations
- Responsive Design
